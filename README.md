# Прототип CRM-системы для оптимизации продаж в ритейле

## Описание проекта
Разработан прототип CRM-системы для розничной торговли, реализующий ключевой функционал для оптимизации процесса продаж.

## Реализованный функционал
- Управление клиентской базой (добавление, просмотр)
- Учёт продаж по категориям товаров
- Автоматический расчёт KPI (LTV, средний чек, конверсия, общая выручка)
- Визуализация продаж по категориям (график Plotly)
- Хранение данных в SQLite

## Технологии
- Python 3.9+
- Google Colab
- SQLite
- Pandas
- Plotly
- ipywidgets

## Установка

### Локальная установка
1. Клонировать репозиторий:
   ```bash

   cd crm-retail-prototype

## Запуск прототипа
Способ 1: Google Colab
1. Открыть Google Colab (colab.research.google.com)
2. Загрузить файл crm_prototype.ipynb
3. Запустить все ячейки (Runtime -> Run all)
4. Работать с системой через выпадающее меню

Способ 2: Локальный запуск
1. Установить Python 3.10+

## Документация кода
Код содержит полную документацию:
Docstrings - описание каждого класса и функции на русском языке
Аннотации типов - указание типов аргументов и возвращаемых значений
Комментарии - пояснения к каждой логической части

## Основные классы
Класс	                     Описание
DatabaseManager	          Управление базой данных (подключение, создание таблиц, добавление данных)
KPICalculator	             Расчёт ключевых показателей эффективности (LTV, средний чек, конверсия)
CRMInterface	             Пользовательский интерфейс (меню, формы, отображение данных)
